## 🛠️ Installation & Local Development  

### 1. Clone the Repository  
```bash
git clone https://github.com/Josh0007-sunday/FlowThriviaGame.git
```
### 2. Change Redirectory  
```bash
cd FlowThriviaGame
```
### 3. Install Package  
```bash
npm install
```
### 4. Run  
```bash
npm run dev
```
